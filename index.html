Frequently Asked Questions

What is WCAG contrast ratio? ▼

WCAG (Web Content Accessibility Guidelines) defines minimum contrast ratios to ensure text is readable by people with visual impairments. AA requires 4.5:1 for normal text, while AAA requires 7:1.

Can I export my test results? ▼

Yes! Pro users can generate professional PDF reports with all test results, color values, and compliance status. Perfect for client deliverables and accessibility audits.

Does it work with my design tools? ▼

Absolutely. Copy/paste hex codes from Figma, Sketch, Adobe XD, or any design tool. Our API also integrates with CI/CD pipelines for automated testing.
